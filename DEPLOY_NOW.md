# 🚀 Deploy Your Website NOW!

## ⚡ **FASTEST METHOD: GitHub Pages (5 Minutes)**

### **Step 1: Enable GitHub Pages**

1. **Go to your repository:**
   ```
   https://github.com/prajwalpatilPp1/Sanrakshan-India
   ```

2. **Click "Settings"** (top right of the page)

3. **Click "Pages"** (left sidebar, under "Code and automation")

4. **Under "Build and deployment":**
   - Source: Select **"Deploy from a branch"**
   - Branch: Select **"main"** and **"/ (root)"**
   - Click **"Save"**

5. **Wait 2-3 minutes** for deployment

6. **Your website will be live at:**
   ```
   https://prajwalpatilpp1.github.io/Sanrakshan-India/
   ```

### **Step 2: Verify Deployment**

1. Refresh the Pages settings page
2. You'll see a green checkmark and your URL
3. Click the URL to visit your live website!

## 🌐 **Want a Custom Domain?**

Instead of `prajwalpatilpp1.github.io/Sanrakshan-India/`
Use: `www.sanrakshanindia.org`

**Steps:**

1. Buy a domain (GoDaddy, Namecheap, etc.)
2. In GitHub Pages settings, add your custom domain
3. In your domain registrar, add these DNS records:

```
Type: A, Name: @, Value: 185.199.108.153
Type: A, Name: @, Value: 185.199.109.153
Type: A, Name: @, Value: 185.199.110.153
Type: A, Name: @, Value: 185.199.111.153
Type: CNAME, Name: www, Value: prajwalpatilpp1.github.io
```

4. Wait 24-48 hours
5. Enable HTTPS in GitHub Pages
