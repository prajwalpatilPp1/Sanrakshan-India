# ⚡ Cloudflare Pages - Quick Start (5 Minutes)

## 🚀 **Deploy Your Website on Cloudflare NOW!**

---

## **STEP 1: Sign Up (2 minutes)**

1. **Go to:** https://dash.cloudflare.com/sign-up

2. **Choose sign-up method:**
   - ⭐ **Recommended:** Sign up with GitHub (1 click!)
   - Or use email + password

3. **Verify email** (if using email signup)

4. **You're in!** Welcome to Cloudflare Dashboard

---

## **STEP 2: Deploy from GitHub (3 minutes)**

1. **In Cloudflare Dashboard:**
   - Click **"Workers & Pages"** (left sidebar)
   - Click **"Create application"** (blue button)
   - Click **"Pages"** tab
   - Click **"Connect to Git"**

2. **Connect GitHub:**
   - Click **"Connect GitHub"**
   - Click **"Authorize Cloudflare"**
   - Select **"Only select repositories"**
   - Find and select: **"Sanrakshan-India"**
   - Click **"Install & Authorize"**

3. **Configure Your Project:**
   ```
   Project name: sanrakshan-india
   Production branch: main
   Framework preset: None
   Build command: (leave empty)
   Build output directory: /
   Root directory: (leave empty)
   ```

4. **Click "Save and Deploy"**

5. **Wait 2-3 minutes** ⏳

6. **DONE!** Your site is live! 🎉

---

## **STEP 3: Get Your URL**

Your website is now live at:
```
https://sanrakshan-india.pages.dev
```

**Click the URL and test your website!**

## 🌐 **OPTIONAL: Add Custom Domain**

### **Want sanrakshanindia.org instead of .pages.dev?**

**Option 1: Buy through Cloudflare (Easiest)**

1. In dashboard, click **"Domain Registration"**
2. Search: **sanrakshanindia.org**
3. Purchase (~$10/year)
4. Go to your Pages project
5. Click **"Custom domains"** → **"Set up a custom domain"**
6. Enter: `sanrakshanindia.org`
7. Click **"Continue"**
8. **Done!** DNS auto-configured ✅

**Option 2: Use existing domain**

1. In dashboard, click **"Websites"** → **"Add a site"**
2. Enter your domain
3. Choose **"Free"** plan
4. Update nameservers at your registrar
5. Wait 24 hours
6. Add to Pages project
